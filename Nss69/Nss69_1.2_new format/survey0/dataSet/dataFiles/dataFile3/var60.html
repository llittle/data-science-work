File: Block - 4 Particulars of living facilities - drinking water, bathroom, sanitation etc - level 4

Description
To tackle the problem of flies/mosquitoes, households may take measures like fixing net in the windows, preventing water from accumulating in the surrounding, use of smoke, use of mosquito repellents, etc. If the household took such measures during last 365 days to tackle the problem of flies/mosquitoes, code will be 1 and if no such measure was taken, code 2 will be recorded. For the households, which had no problem of flies/mosquitoes and no measure was taken by the household, code 3 will be applicable.
